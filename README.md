## 🧠💡 About the Project

**AI Resume Builder** is not just another resume tool —
it’s your **personal AI assistant** for crafting professional, modern, and impactful resumes in minutes.

## 🔥⚡ Features

✨ Guided resume creation (no confusion)
🤖 AI-powered content generation
🎯 Smart suggestions for skills & experience
👀 Real-time preview (what you see = what you get)

## 🚀⚡ Quick Start

```bash
# Clone repo
git clone <repo-link>
cd ai-resume-builder

# Install dependencies
cd client && npm install
cd ../server && npm install

# Run project
cd client && npm start
cd ../server && npm run dev
```
